{# Shared head fragment — meta + font + CDN preconnects. Included inside every template's via {% include '_head.html' %}. Preconnects: open the TCP+TLS connection to each CDN before any of their scripts/styles are requested. Saves ~50-200ms per origin on first-visit cold load — the request can fire immediately instead of waiting on DNS+handshake. crossorigin attribute matches the eventual fetch credentials mode (gstatic/jsdelivr/unpkg/mapbox are all anonymous/CORS). Fonts: preconnects + a single with both families. Loading here (not via @import in base.css) lets the browser fetch in parallel with base.css instead of serially. `display=swap` keeps the system fallback visible while webfonts download — no flash of invisible text. #} {# Running version — read by whatsnew.js for the header chip + update banner. #} {# Top progress strip — shows the cold-load sidebar-cache build progress. #} {# Manual track upload — injects the header "Upload" button on every page. #} {# What's New — injects the version chip + shows the post-update banner. #} {# Guide — injects a persistent "Guide" link into the header nav. #}
